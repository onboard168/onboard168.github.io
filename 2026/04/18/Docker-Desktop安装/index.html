Docker Desktop 安装

发表于 2026-04-18

使用命令提示符 (CMD) 工具进行安装。

在 Windows 上将 Docker Desktop 安装到 D 盘并解决权限问题，其实是一个”曲线救国”的过程。因为 Docker 的官方安装包默认强制安装在 C:\Program Files\Docker\Docker，且没有提供直接的选择路径界面。

我们可以通过目录软链接（Symbolic Link）的方式，骗过安装程序，让它以为装在 C 盘，实则文件存储在 D 盘。

第一步：准备工作与解决权限问题

在操作 D 盘时，如果提示”无权限创建目录”，通常是因为 D 盘根目录受系统保护，或者你当前的账户不是管理员。

1. 以管理员身份运行终端

按下 Win + X，选择终端管理员或命令提示符（管理员）。

只有在管理员模式下，你才有权在 D 盘根目录创建文件夹，并建立系统级软链接。

第三步：安装 Docker Desktop

  1. 从 Docker 官网 下载安装包
  2. 正常双击运行安装程序
  3. 关键点：安装过程中不要修改任何路径（反正它也不让你改），直接点击安装
  4. 安装程序会将文件写入 C:\Program Files\Docker，但由于我们之前做了软链接，这些文件会自动流向 D 盘

3. 卸载后再安装

如果之前安装过 Docker，建议先完全卸载：

  1. 控制面板卸载 Docker Desktop
  2. 删除 C:\Program Files\Docker（如果是软链接，先删除链接）
  3. 删除 D:\Software\Docker 中的所有内容
  4. 重新按照上述步骤安装
